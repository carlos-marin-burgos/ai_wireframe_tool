Step 1: Check if React app is accessible

Try opening the React app at: http://localhost:5173

If you see a blank page, proceed with the debugging steps below.

Step 2: Open Browser Developer Tools

Press F12 or right-click and select "Inspect Element"

Go to the Console tab to see JavaScript errors

Step 3: Common Issues to Check

  • JavaScript Errors: Look for red error messages in console
  • Missing Dependencies: Check for import/module errors
  • Drag & Drop Component Errors: Specifically look for errors related to:
    • LinkableWireframe component
    • EnhancedDragWireframe component
    • DragPalette component
  • CSS Issues: Check if styles are loading properly
  • React Render Errors: Look for component mounting/rendering failures

Step 4: Network Tab Check

Go to the Network tab and reload the page

Look for any failed requests (shown in red)

Common issues: Missing CSS files, JavaScript bundles, or API calls

Step 5: Elements Tab Check

Go to the Elements tab

Check if the React root element exists: <div id="root">

If the div is empty, React is not mounting properly

Step 6: Check for Drag & Drop Related Errors

Since the issue started after drag & drop changes, look for:

  • Uncaught TypeError in drag event handlers
  • Missing imports for drag components
  • CSS conflicts with drag styles
  • State management issues in drag components

Step 7: Test Local API Connection
